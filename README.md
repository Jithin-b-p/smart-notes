## Local Development

### Prerequisites

- Node.js v20+
- pnpm v9+
- Docker Desktop

### Setup

```bash
# Clone the repository
git clone https://github.com/Jithin-b-p/smart-notes.git
cd smart-notes

# Install dependencies
pnpm install

# Start local services (Postgres + Redis)
docker compose up -d
```
